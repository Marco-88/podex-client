# Codex Sandbox Clone

Just got my beta key and now I wanna learn by cloning OpenAI's sandbox

## Prerequisites

- node installed
- openai api key

## Setup

Create `.env` file in project root and add:

```env
VITE_OPENAI_API_KEY=yourapikey
```

Enter in your command line termial

```shell
npm install
npm run start
```
